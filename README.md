Retail Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing retail sales data using MySQL and Power BI. The goal is to derive meaningful business insights from sales transactions and 

present them through an interactive dashboard.

## Project Workflow

Created and managed the database in MySQL.
   
Imported the retail sales dataset into MySQL.

Performed data cleaning and exploratory data analysis using SQL queries.

Solved business-related questions using SQL.

Connected MySQL data to Power BI.

Built an interactive dashboard to visualize key business insights.

## Dashboard Features

* Sales Trend Analysis
  
* Customer Analysis
  
* Category Performance Analysis
  
* Revenue Contribution Analysis
  
* Top Customer Analysis
  
* Interactive slicers for Year, Month, Category, and Gender

## Tools and Technologies Used

* MySQL
  
* SQL
  
* Power BI
  
* DAX

## Key SQL Operations Performed

* Database and table creation
  
* Data cleaning
  
* Data exploration
  
* Aggregations and grouping
  
* Business query analysis

## Dashboard Pages

### 1. Overview

Provides a high-level summary of revenue, customers, average order value, and transactions.

### 2. Sales Trend

Analyzes monthly sales trends, sales by day, and sales by hour.

### 3. Category Analysis

Shows category-wise revenue, contribution percentage, and units sold.

### 4. Customer Analysis

Analyzes customer demographics, purchase frequency, top customers, and gender-based revenue contribution.
